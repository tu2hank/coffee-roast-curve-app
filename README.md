# 咖啡試烘曲線比較表

這個資料夾集中放置咖啡試烘曲線比較表相關檔案。

## 主要檔案

- `index.html`：咖啡試烘曲線比較表主頁，可直接用瀏覽器開啟。
- `dev-server.cjs`：本機預覽用的小型伺服器。

## 目前功能

- 同一支豆子的試烘目標與假設
- 支援 3 鍋不同烘法比較
- 每鍋獨立記錄烘法設定、重要階段、每 30 秒一筆的詳細數據
- 多鍋烘焙曲線疊圖比較
- 每鍋摘要：損失率、發展時間、最高溫、杯測判斷
- 杯測比較表
- 最適合烘法與下次調整結論
- CSV 比較匯出
- 列印 / 存為 PDF
- 瀏覽器本機自動保存
